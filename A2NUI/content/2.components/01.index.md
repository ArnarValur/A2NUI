## Overview

The A2NUI library implements every component in the A2UI v0.10 standard catalog. Each A2UI component is mapped to one or more Nuxt UI widgets.

## Display Components

| Component                              | Description                                       | A2UI → Nuxt UI       |
| -------------------------------------- | ------------------------------------------------- | -------------------- |
| [Text](/components/text)               | Headings (h1–h5), body, and caption text variants | `<h1>`–`<h5>`, `<p>` |
| [Image](/components/image)             | Responsive images with fit and variant sizing     | `<img>`              |
| [Icon](/components/icon)               | A2UI icon names mapped to Lucide icons            | `UIcon`              |
| [Video](/components/video)             | Native video player with controls                 | `<video>`            |
| [AudioPlayer](/components/audioplayer) | Audio player with optional description label      | `<audio>`            |

## Layout Components

| Component                      | Description                                             | A2UI → Nuxt UI         |
| ------------------------------ | ------------------------------------------------------- | ---------------------- |
| [Row](/components/row)         | Horizontal flex container with gap, justify, and align  | `flex-row`             |
| [Column](/components/column)   | Vertical flex container with gap, justify, and align    | `flex-col`             |
| [List](/components/list)       | Scrollable container — vertical or horizontal direction | `flex + overflow-auto` |
| [Card](/components/card)       | Card wrapper with optional title header                 | `UCard`                |
| [Tabs](/components/tabs)       | Tab container with dynamic child content per tab        | `UTabs`                |
| [Divider](/components/divider) | Visual break between sections                           | `USeparator`           |

## Interactive Components

| Component                                  | Description                                      | A2UI → Nuxt UI              |
| ------------------------------------------ | ------------------------------------------------ | --------------------------- |
| [Button](/components/button)               | Button with primary, outline, and ghost variants | `UButton`                   |
| [TextField](/components/textfield)         | Text input — short or long text variant          | `UInput` / `UTextarea`      |
| [CheckBox](/components/checkbox)           | Checkbox with label and boolean value            | `UCheckbox`                 |
| [ChoicePicker](/components/choicepicker)   | Radio group or checkbox list based on variant    | `URadioGroup` / `UCheckbox` |
| [Slider](/components/slider)               | Range input with min/max and label               | `USlider`                   |
| [DateTimeInput](/components/datetimeinput) | Date, time, or datetime-local input              | `UInput` (type=date/time)   |

## Overlay Components

| Component                          | Description                             | A2UI → Nuxt UI |
| ---------------------------------- | --------------------------------------- | -------------- |
| [Modal](/components/modal)         | Overlay dialog with trigger and content | `UModal`       |
| [Slideover](/components/slideover) | Panel that slides in from a screen edge | `USlideover`   |
